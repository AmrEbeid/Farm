## Step 1 — pre-flight (read-only; re-run right before applying)

Run in the Supabase SQL editor (or `psql`). **Every row must be `0`** except the migration-head row,
which must read `20260622000048`. (This exact query passed clean on 2026-06-26 — re-run to catch any
data drift before you apply.)

```sql
select 'CHK 0054 plan_operations.est_cost<0'        as check_name, count(*)::text as cnt from public.plan_operations where est_cost < 0
union all select 'CHK 0054 pmr.qty<0',                count(*)::text from public.plan_material_requirements where qty < 0
union all select 'CHK 0054 plr.count<0',             count(*)::text from public.plan_labor_requirements where count < 0
union all select 'CHK 0054 plr.days<0',              count(*)::text from public.plan_labor_requirements where days < 0
union all select 'CHK 0056 pri.qty<=0',              count(*)::text from public.purchase_request_items where qty <= 0
union all select 'CHK 0058 plan_op.status not in vocab', count(*)::text from public.plan_operations where status not in ('planned','approved','reserved','ready','in_progress','done','blocked','abandoned','skipped')
union all select 'XORG 0061 pri.item cross-org',     count(*)::text from public.purchase_request_items pri left join public.inventory_items it on it.id=pri.item_id and it.org_id=pri.org_id where pri.item_id is not null and it.id is null
union all select 'XORG 0061 pmr.item cross-org',     count(*)::text from public.plan_material_requirements pmr left join public.inventory_items it on it.id=pmr.item_id and it.org_id=pmr.org_id where pmr.item_id is not null and it.id is null
union all select 'XORG 0063 pri.supplier cross-org', count(*)::text from public.purchase_request_items pri left join public.suppliers s on s.id=pri.supplier_id and s.org_id=pri.org_id where pri.supplier_id is not null and s.id is null
union all select 'XORG 0063 inv.pref_supplier xorg', count(*)::text from public.inventory_items i left join public.suppliers s on s.id=i.preferred_supplier_id and s.org_id=i.org_id where i.preferred_supplier_id is not null and s.id is null
union all select 'TRG pre-exist (must be 0)',         count(*)::text from pg_trigger where tgname in ('audit_supplier','audit_pr_item','audit_inventory_item','audit_responsibility','audit_people')
union all select 'INFO prod migration head',          (select max(version) from supabase_migrations.schema_migrations)
order by 1;
```
