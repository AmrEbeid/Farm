# دليل استخدام نظام تشغيل المزارع

مرحبًا بك في دليل استخدام **نظام تشغيل المزارع** (مزارع عبيد).

هذا الدليل مكتوب لموظفي المزرعة. لا يحتاج إلى خبرة في الحاسب. الجُمل قصيرة وبسيطة، والخطوات بالترتيب.

التطبيق مصمَّم للهاتف المحمول أولًا، وواجهته باللغة العربية من اليمين إلى اليسار.

## أقسام الدليل

1. **[البداية وتسجيل الدخول](01-getting-started.md)**
   كيف تدخل إلى التطبيق بالبريد الإلكتروني وكلمة المرور، وماذا يرى كل دور، والتنقّل في التطبيق على الهاتف.

2. **[تغطية المخزون وطلبات الشراء](02-stock-coverage-and-purchase-requests.md)**
   متابعة كميات المخزون، ومتى تنشئ طلب شراء جديد.

3. **[الموافقات والميزانية](03-approvals-and-budget.md)**
   كيف تُراجَع طلبات الشراء وتُعتمَد، وكيف تُتابَع الميزانية.

4. **[الاستلام والتنفيذ](04-receipts-and-execution.md)**
   استلام الأصناف وتسجيل العمليات في الميدان.

5. **[التقارير ولوحات التحكم](05-reports-and-dashboards.md)**
   عرض ملخّصات العمل والمخزون والمصروفات، وقراءة تقرير المخطط مقابل الفعلي.
